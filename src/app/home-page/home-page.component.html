Sorting Algorithms

A Sorting Algorithm is used to rearrange a given array or list elements according to a comparison operator on the elements. The comparison operator is used to decide the new order of element in the respective data structure.

The sorting algorithms we have visualised here along with their complexities are as follows:-

  • Quick Sort: O(n2) for worst case, O(nLogn) for best and average case
  • Heap Sort: O(nLogn)
  • Merge Sort: O(nLogn)
  • Bubble Sort: O(n2) for worst case, O(n) for best case
  • Selection Sort: O(n2)
  • Insertion Sort: O(n2)
